# **App Name**: MoneyWise

## Core Features:

- Income and Expense Tracking: Allow users to input their daily income and expenses with details such as amount, date, and notes.
- Custom Categories: Enable users to create and manage custom categories for both income and expenses to better organize their transactions.
- Transaction History: Display a chronological list of all recorded transactions with filtering and sorting options.
- Spending Insights Tool: Utilize AI to analyze transaction patterns and provide insights into spending habits or potential areas for saving.
- Budget Overview: Show a summary of current finances, including total income, total expenses, and remaining balance for a selected period.

## Style Guidelines:

- The primary color, a deep and trustworthy blue (#336699), grounds the financial data, conveying stability and professionalism. It contrasts clearly against a bright background.
- A very light, desaturated blue-gray background (#F0F2F4) offers a clean, airy canvas that is easy on the eyes and prevents visual clutter, allowing content to stand out.
- An accent color of vibrant violet (#E052E0) adds a contemporary and sophisticated touch, used sparingly for calls to action, highlights, and important visual cues.
- Headline and body font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, ensuring clarity across all financial data displays and textual content.
- Use clear, universally recognizable icons related to financial transactions (e.g., wallet, pie chart, plus/minus symbols) to intuitively guide users.
- Adopt a clean, structured layout with a card-based system for displaying transaction summaries and category breakdowns. Ensure responsive design for seamless experience across devices.
- Implement subtle, swift animations for state changes and data updates, such as transaction additions or filter applications, to provide fluid user feedback without distraction.
